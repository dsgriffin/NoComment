URL/URL pattern syntax (important!):

1. The wildcard selector * can be used to match anything/everything, depending on where you use it.

2. If the URL/URL pattern you want to target includes a prefix such as www. (e.g. www.example.com), then you need to include this prefix.

YouTube example: www.youtube.com/* targets all YouTube videos.

X example: x.com/*/status/* targets all X statuses.

Reddit example: www.reddit.com/r/news/* targets any posts under the r/news subreddit.

About

NoComment is a simple Chrome extension for hiding user-generated comments on the web.

I'd appreciate a rating in the Google store, along with any constructive criticism you have! Alternatively, feel free to open an issue on GitHub if you've discovered any major problems with the extension.

Defaults & General Settings

On a fresh installation, NoComment does not block any comments.

If you'd like to block comments on specific websites/routes only, you can add those websites/routes to your Blocklist.

If instead you prefer to block comments on ALL websites by default, then you can select the option Yes (Except Allowlist) & add any particular websites/routes to your Allowlist if you don't want to block comments on them.

You'll notice that, by default, comments are "collapsed"; as in, the space they initially took up is completely removed.

If you prefer, you can change this to "hidden"; the hidden option also hides the comments themselves, but keeps the space these comments usually take up when visible.

Allowlist/Blocklist URL Pattern Syntax

This is discussed (with example use-cases) at the top of the Allow List/Block List page.

It is vital that you remember to include the www. prefix if the website you're using displays it.

Just as important is to include the wildcard selector * in the correct place.

Uninstall NoComment

First, navigate to the Extensions menu; either type chrome://extensions into the address bar or go to Chrome Menu > Tools > Extensions.

Then, click the recycle bin icon that is next to the NoComment extension.
