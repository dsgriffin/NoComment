URL/URL pattern syntax (important!):

1. The wildcard selector * can be used to match anything/everything, depending on where you use it.

2. If the URL/URL pattern you want to target includes a prefix such as www. (e.g. www.example.com), then you need to include this prefix.

Example 1: www.youtube.com/*

Targets: all youtube videos

Example 2: www.reddit.com/r/news/*

Targets: any posts under the r/news subreddit

About

NoComment is a simple Chrome extension for hiding user-generated comments on the web.

I'd appreciate a rating in the Google store (along with any constructive criticism you have). If the application doesn't function correctly on a particular website, let me know by opening an issue on the Github repo.

Defaults & General Settings

On a fresh installation, NoComment does the following:

  • Blocks comments on all websites where comments are successfully detected.
  • Visually, these comments are "collapsed"; as in, the space they initially took up is completely removed.

Both of these settings can be changed:

  • NoComment can either block all comments on all websites where comments are detected (except on website routes that are in the Allowlist), or block no comments on all pages (except on website routes that are in the Blocklist).
  • Comments can either be "collapsed" (previously explained) or "hidden" - the hidden option also hides the comments themselves, but keeps the space these comments usually take up when visible.

Allow List/Block List

  • The Allow List is where users can store URLs/URL patterns they wish to be exempt from the comment blocking process that occurs when the "Block all comments on all pages" setting is active.
  • The Block List is the opposite - it is where users can store URLs/URL patterns they wish to block comments on when the "Block no comments on all pages" setting is active.

URL/URL Pattern Syntax

This is discussed (with example use-cases) at the top of the Allow List/Block List page. Be sure to read it, especially if plan on using the URL pattern wildcard (*) selector.

Uninstall NoComment

First, navigate to the Extensions menu; either type chrome://extensions into the address bar or go to Chrome Menu > Tools > Extensions.

Then, click the recycle bin icon that is next to the NoComment extension.
