URL/URL pattern syntax (important):

  • If a leading protocol (e.g http://, https://) is not specified, all protocols will be targeted.

    Example: www.example.com Targets: http://www.example.com, https://www.example.com.

  • The wildcard selector * can be used to match anything/everything, depending on where you use it.

    Example 1: http://www.example.com/* Targets: http://www.example.com/about.html, http://www.example.com/index/secondDirectory.html.

    Example 2: https://example.*/about Targets: https://example.co.uk/about, https://example.org/about.

  • If the URL/URL pattern you want to target includes a prefix such as www. (e.g. www.example.com), then you need to include this prefix.

    Example 1: example.com/hi.txt Targets: http://example.com/hi.txt, https://example.com/hi.txt.

    Example 2: www.example.com/hi.txt Targets: http://www.example.com/hi.txt, https://www.example.com/hi.txt.

General

NoComment is a simple Chrome extension for hiding user-generated comments on the web.

I would appreciate a rating in the Google store along with any constructive criticism you have. If the application doesn't function correctly on a particular website, let me know; preferably by opening an issue on the Github repo.

If you'd like to donate (thank you *so* much) you can send it to paypal.me/dsgriffin

Purpose

There are various reasons why someone may wish to hide comments on all/specific sites, including:

  • Productivity - sometimes you can find yourself spending a lot of time reading comments and not really gaining anything useful from it.
  • Noise - sometimes comments generate unnecessary noise to an article (e.g. off-topic discussions).
  • Trolls - *Tips Fedora*
  • Too much negativity on the Internet - sadly, it isn't uncommon to come across ignorant/hurtful/abusive comments online.

Regardless of whether or not a user's reason for using this extension is on the list above; I hope it improves their internet experience in some way.

How To Use/Features Explained

Default And General Settings

On a fresh installation, NoComment does the following:

  • Blocks comments on all websites where comments are successfully detected.
  • Visually, these comments are "collapsed" - as in the space they initially took up is completely removed.

Both of these settings can be changed:

  • NoComment can either block all comments on all pages by default (except those on websites that match an entry in the Allow List), or block no comments on all pages by default (except those on websites that match an entry in the Block List).
  • Comments can either be "collapsed" (previously explained) or "hidden" - the hidden option also hides the comments themselves, but keeps the space these comments usually take up when visible.

Allow List/Block List

Purpose

  • The Allow List is where users can store URLs/URL patterns they wish to be exempt from the comment blocking process that occurs when the "Block all comments on all pages" setting is active.
  • The Block List is the opposite - it is where users can store URLs/URL patterns they wish to block comments on when the "Block no comments on all pages" setting is active.

URL/URL Pattern Syntax

This is discussed (with example use-cases) at the top of the Allow List/Block List page. Be sure to read it, especially if plan on using the URL pattern wildcard (*) selector.

List Actions

  • Add New - Add a new URL/URL pattern to the list.
  • Modify Selected (one) - Modify the currently selected URL/URL pattern. Click on a row to select it. This action is only allowed when exactly one row is selected.
  • Remove Selected (one/multiple) - Remove the currently selected URL/URL patterns. Click on rows to select them. This action can be used to remove one or more list entries at a time.
  • Remove All - Remove all URL/URL pattern from the list.

Browser Icon

This will only show on pages where the blocking of comments is in effect. Clicking on the icon will show you some information about NoComment, as well as a quick-link to the options page (the same one you're reading right now!), where you can change your settings.

Uninstall NoComment

If you would like to completely uninstall this extension, you can do so via.

  • Navigating to the Extensions menu - either type chrome://extensions into the address bar or go to Chrome Menu > Tools > Extensions.
  • Click the recycle bin icon that is next to the NoComment extension.

If you do decide to uninstall, I would appreciate it if you let me know why via. rating feedback in the Chrome store, so I know what needs improving.
